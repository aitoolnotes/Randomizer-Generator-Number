# 🎲 Randomizer Generator Number

Ein kostenloser, benutzerfreundlicher Zufallszahlengenerator Tool mit vollständiger Dokumentation und optimiert für mobile Geräte.

## 📋 Features

- ✨ **Einfach zu bedienen** - Intuitive Benutzeroberfläche
- 📱 **Mobil-optimiert** - Responsive Design für alle Geräte
- ⚡ **Schnell** - Sofortige Ergebnisse ohne Wartezeit
- 🔒 **Datenschutz** - Keine Datenerfassung oder Tracking
- 🎯 **Flexibel** - Generieren Sie eine oder mehrere Zahlen gleichzeitig
- 🌍 **Mehrsprachig** - Auf Deutsch verfügbar
- 📋 **Kostenlos** - Keine versteckten Gebühren oder Registrierung
- ♿ **Barrierearm** - Zugänglich für alle Benutzer

## 🚀 Schnellstart

### Lokal testen

1. **Repository klonen:**
   ```bash
   git clone https://github.com/[USERNAME]/randomizer-generator-number.git
   cd randomizer-generator-number
   ```

2. **Lokal starten:**
   - Öffnen Sie einfach `index.html` in Ihrem Browser
   - Oder nutzen Sie einen lokalen Server:
   ```bash
   python -m http.server 8000
   # oder mit Node.js
   npx http-server
   ```

3. **Im Browser öffnen:**
   ```
   http://localhost:8000
   ```

### Deployment auf Cloudflare Pages

1. **Push zu GitHub:**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Cloudflare Pages verbinden:**
   - Gehen Sie zu [Cloudflare Pages](https://pages.cloudflare.com/)
   - Klicken Sie auf "Create a project"
   - Verbinden Sie Ihr GitHub-Konto
   - Wählen Sie dieses Repository
   - Build-Einstellungen:
     - **Framework preset:** None
     - **Build command:** (leer lassen)
     - **Build output directory:** (leer lassen)
   - Klicken Sie auf "Save and Deploy"

3. **Custom Domain (optional):**
   - Gehen Sie zu Domain Settings
   - Fügen Sie Ihre eigene Domain hinzu
   - Aktualisieren Sie die DNS-Einstellungen
